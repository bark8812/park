## :rocket: Java Swing으로 만든 영화예매프로그램 <br/>

- KG 그룹에서 영화산업에 뛰어들면서 영화예약 관리프로그램 개발 필요하다는 상황을 가정하여 영화예약 프로그램을 만들어 보기로 함. 
- 네이버 영화예매를 벤치마킹하여 개발을 함 
- 관리자는 없이 단순히 예매, 예매확인, 예매취소만 가능한 프로그램
